Thanks for using this template!

Here are a few things to help you quickly change the template to suit your needs.

1. This document uses "Google Fonts." The font used in the design view is different to the one used in the preview, so use Preview in Browser to see how it looks. You can install the font used "Nixie One", which can be found here: <http://www.google.com/fonts>, and select it as the screen font in the Edit>Font Sets dialog. If you want to read a guide on how to set up a Google Web Font, check out this KnowledgeBase Article <http://www.softpress.com/kb/questions/276>.

2. Changing Social Media links

Go to Edit>URLs.

Select the Facebook URL and click "Edit".

In the "Change to URL" field, paste in the web address of your Facebook group/page and click "OK".

Repeat this for any others you want to use (If you don't want to use these Social Icons, you can delete them by selecting the Master Page, selecting the icons you want to remove and then deleting them).

3. If you want to change any of the colors of this template you can do so via the Edit>Colors dialog.

4. In order to get your Contact Form to send to your email address, do the following:

Select the "Contact" page.

Open the Actions palette (Window>Actions), you should be able to see the Send Form Action in the Actions palette. If not, then make sure you're selecting the Page and not an Item on the Page.

Look in the Actions palette for "Recipient Address" and change "youremailaddresshere@domain.com" to your email address that you'd like to use.

5. Begin changing the content of the site to contain what you want!

If you have any problems, email us at <support@softpress.com> with a copy of your file.
